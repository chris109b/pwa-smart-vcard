Informationen

Diese Applikation steht kostenlos zur Verfügung. Die Applikation dient dazu eine geschäftliche und eine private Visitenkarte auf dem Smartphone anzuzeigen.

Alle eingegeben Daten werden offline auf dem Gerät und in dem Webbrowser gespeichert, in dem sie eingegeben wurden.

Die Informationen der gespeicherten Visitenkarten werden im V-Card-Format kodiert und als QR-Code angezeigt. Dieser kann von einer entsprechenden App wie z.B. der "Kamera"-App auf dem iPhone oder der "Binary Eye"-App von "Markus Fisch" auf einem Android-Smartphone gelesen werden. Die Informationen werden dabei als Eintrag für das Adressbuch erkannt und können direkt in selbiges übernommen werden.

Die Daten werden auch in Textform angezeigt. Sollte also jemand nicht in der Lage sein, einen QR-Code zu scannen, können die Daten manuell übertragen werden.

Felder, die in den Einstellungen leer gelassen werden, werden ignoriert. Möchte man also z.B. seine private Adresse nicht auf der privaten Visitenkarte haben, lässt man "Straße, Hausnummer", "Postleitzahl" und "Ort" einfach leer. Gleiches gilt für Telefonnummern und alle anderen Felder.

Als private Homepage lässt sich auch ein Link auf das eigene Soziale Netzwerk Profil verwenden. Das funktioniert z.B. mit einem Facebook-Profil.

Installation

Diese Anwendung ist eine sogenannte "Progressive Web App". Sie kann wie eine reguläre App aus dem App-Store benutzt werden.

Für die Installation auf iOS-Geräten (iPhone, iPad und iPod Touch) öffnet man diese Anwendung als Web-Seite in Safari.

  • Dort tippt man auf das "Teilen"-Symbol (mittig, unten).
  • In der Teilen-Funktion wählt man "Zum Home-Bildschirm".
  • Im nachfolgenden Bestätigungsdialog ließe sich der Name ändern und man wählt "Hinzufügen" (rechts, oben).
  • Die Installation ist damit abgeschlossen.

Auf einem Android-Gerät mit Chrome-Browser öffnet man die Anwendung als Web-Seite in Chrome.

  • Dort öffnet man das Menü (recht, oben, drei Punkte untereinander).
  • Im Menü wählt man die Option "Zum Startbildschirm hinzufügen".
  • Im nachfolgenden Bestätigungsdialog tippt man auf "Hinzufügen".
  • Die Installation ist damit abgeschlossen.

Die Installation funktioiert auf anderen Geräten ähnlich.
